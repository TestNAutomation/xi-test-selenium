Calls `$func` repeatedly until it returns a non-null value or `$timeout` seconds have elapsed.

Throws a SeleniumException on timeout. Exceptions from $func pass through.

This is used internally to implement the `waitFor*` methods, but can also be used by client code.

Parameters:
callable$funcA callback to invoke.
int | float$timeoutThe number of seconds to try for.
string$timeoutMsgThe message of the SeleniumException to throw on timeout.
Returns:
The return value of `$func`
Exceptions:
SeleniumExceptionif `$func` only returns nulls and the timeout elapses.
Exceptionif `$func` throws it.
