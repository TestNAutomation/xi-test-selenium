Detailed Description

Injects a WebDriver into a test suite (or individual test) and properly disposes of it afterwards.

It calls setUpWebDriver() for any test implementing WebDriverTest. This is done recursively if the test is an IteratorAggregate (e.g. a PHPUnit test suite).
